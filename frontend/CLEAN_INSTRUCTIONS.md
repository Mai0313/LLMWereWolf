# 🛠️ 清理版使用指南

## 快速启动

```bash
cd frontend
chmod +x quick-start.sh
./quick-start.sh
```

访问 http://localhost:3000

## 🎯 企业API已集成完成

### ✅ 集成的模型
- **Llama-3.3-70B** (备用服务器)
- **Nemotron-49B** (主服务器)
- **Devstral-Small** (主服务器)
- **Gemma-3-27B** (主服务器)
- **GPT-OSS-120B** (主服务器)

### ✅ API 端点
- 主服务器: `http://100.80.20.5:4000/v1`

## 🔧 配置说明

所有API配置已设置在 `.env` 文件中：
```bash
REACT_APP_LLM_SERVER_1=http://100.80.20.5:4000/v1
REACT_APP_LLM_SERVER_2=http://100.82.5.110:30001
REACT_APP_DEFAULT_MODEL=meta-llama/Llama-3.3-70B-Instruct
```

## 🎯 使用步骤

1. **启动前端**: `./quick-start.sh`
2. **打开**: http://localhost:3000
3. **配置游戏**: 点击右上角设置按钮，选择"游戏配置"
4. **配置AI**: 切换到"AI配置"标签页，测试并选择模型
5. **启动游戏**: 保存配置后点击"启动游戏"
