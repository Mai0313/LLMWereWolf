# 🚀 快速入门指南

## 一分钟运行 React UI

### 1. 进入前端目录
```bash
cd frontend
```

### 2. 运行启动脚本
```bash
./start.sh
```

### 3. 访问应用
打开浏览器访问: http://localhost:3000

---

## 🔧 手动安装（如果脚本失败）

### 安装依赖
```bash
npm install
```

### 创建环境变量
```bash
cp .env.example .env
```

### 启动开发服务器
```bash
npm run dev
```

## 🐛 常见问题

### Q: 页面白屏
**A:** 检查控制台错误，确保所有依赖已正确安装

### Q: API连接失败
**A:** 确保后端服务启动，或使用模拟数据模式

### Q: 样式显示异常
**A:** 尝试强制刷新浏览器缓存 (Ctrl+F5)

### Q: 动画卡顿
**A:** 在设置中降低动画质量或关闭特效
